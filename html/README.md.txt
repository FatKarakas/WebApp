# HTML Dosyaları

Bu klasörde web sayfasının yapısını oluşturan HTML dosyaları bulunmaktadır.  

## Açıklama
- `index.html` ana sayfa olarak kullanılır.
- Diğer HTML dosyaları varsa sayfa yapısını destekler.
- Sayfa içeriği, başlıklar, paragraflar, bağlantılar ve temel yapı öğeleri burada bulunur.
- Telif durumu belirsiz resimler yorum satırı ile belirtilmiştir

## Kullanım
1. HTML dosyalarını tarayıcınızda açarak görüntüleyebilirsiniz.
2. CSS ve JavaScript dosyaları ile birlikte kullanıldığında sayfa tam olarak çalışır.
