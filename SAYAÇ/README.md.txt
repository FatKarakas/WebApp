Sayaç Uygulaması (HTML / CSS / JavaScript)
Bu proje, HTML, CSS ve JavaScript kullanılarak geliştirilmiş basit bir sayaç (counter) uygulamasıdır. Kullanıcı, butonlar aracılığıyla sayıyı artırabilir, azaltabilir veya sıfırlayabilir.
Temel web teknolojilerini öğrenmek ve JavaScript ile DOM manipülasyonunu kavramak amacıyla hazırlanmıştır.

Kullanılan Teknolojiler:
HTML5 → Sayfa yapısının oluşturulması
CSS3 → Butonlar ve sayaç görünümünün stillendirilmesi
JavaScript → Sayaç işlemleri ve kullanıcı etkileşimi

Uygulama Özellikleri:
Artırma Butonu: Sayaç değerini 1 artırır
Azaltma Butonu: Sayaç değerini 1 azaltır
Yenile (Reset) Butonu: Sayaç değerini sıfırlar
Hover efekti olan buton tasarımı
Sayaç değeri ekranda büyük ve okunaklı şekilde gösterilir

Öğrenilen Konular:
document.getElementById() kullanımı
JavaScript ile DOM manipulation
onclick olayları
CSS hover efektleri
HTML–CSS–JS dosya entegrasyonu

Ekran Görünümü Açıklaması:
Bu görselde, HTML, CSS ve JavaScript kullanılarak geliştirilen sayaç uygulamasının çalışır hali görülmektedir.
Sayfanın üst kısmında, sayaç değeri büyük ve kalın font ile gösterilmekte olup başlangıç değeri 0’dır. Sayaç değeri, kullanıcı işlemlerine göre anlık olarak güncellenmektedir.

Sayacın altında yatay hizalanmış şekilde üç adet buton bulunmaktadır:
Azalt butonu, sayaç değerini 1 azaltmak için kullanılır.
Yenile butonu, sayaç değerini sıfırlamak için kullanılır.
Artırma butonu, sayaç değerini 1 artırmak için kullanılır.

Butonlar açık renk arka plan, yuvarlatılmış köşeler ve hover (üzerine gelince renk değişimi) efekti ile tasarlanmıştır. Bu sayede kullanıcı etkileşimi görsel olarak desteklenmiştir.
Uygulama, JavaScript ile gerçekleştirilen DOM manipülasyonu sayesinde kullanıcı etkileşimlerine anlık tepki vermektedir.
