# CSS Dosyaları

Bu klasörde web sayfasının stil ve tasarım dosyaları bulunmaktadır.  

## Açıklama
- Sayfa renkleri, yazı tipleri, boyutlar ve konumlandırmalar CSS ile kontrol edilir.
- `style.css` ana stil dosyasıdır.
- Gerekirse ek stil dosyaları buraya eklenebilir.

## Kullanım
1. HTML dosyalarında `<link rel="stylesheet" href="css/style.css">` ile bağlanır.
2. Tarayıcıda sayfayı açtığınızda tasarım ve stiller uygulanır.
