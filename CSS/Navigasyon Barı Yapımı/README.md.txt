Bu proje, web tasarım dersi kapsamında oluşturulmuş bir navigasyon (menü) barı uygulamasıdır.
Proje, kullanıcı dostu bir gezinme menüsü oluşturmayı ve temel HTML, CSS (ve opsiyonel olarak JavaScript) becerilerini pekiştirmeyi amaçlamaktadır.

Özellikler:
Duyarlı (responsive) tasarım
Hover (üzerine gelince renk değişimi) efektleri
Aktif sayfa göstergesi
Menü elemanlarının yatay hizalanması(istenirse dikey olabilir)

Öğrenme Kazanımları:
HTML etiket yapısını kavrama
CSS ile düzenleme, hizalama ve renk uyumu
Temel kullanıcı deneyimi (UX) ilkeleri
Duyarlı (responsive) tasarım mantığı
