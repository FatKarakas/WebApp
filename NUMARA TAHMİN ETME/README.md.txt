NUMARA TAHMİN ETME ALGORİTMASI
Proje Amacı:
Bu proje, kullanıcıdan belirli bir aralıkta sayı tahmini alarak doğru sayıyı bulmasını sağlayan bir JavaScript algoritması içerir. 
Program kullanıcıyı yönlendirir, küçük/büyük uyarıları verir ve doğru tahmin yapıldığında toplam deneme sayısını gösterir.

Kullanılan Teknolojiler:HTML5,JavaScript,(İsteğe bağlı) CSS

Nasıl Çalışır?
-Sayfa açıldığında JavaScript otomatik çalışır.
-Program 1 ile 100 arasında rastgele bir sayı üretir.
-Kullanıcıdan prompt() ile tahmin girmesi istenir.
-Girilen değer kontrol edilir:

Sayı değilse → “Geçerli sayı giriniz”
Aralık dışındaysa → “Geçerli sayı giriniz”
Küçükse → “Sayı çok küçük”
Büyükse → “Sayı çok büyük”
Doğruysa → Doğru tahmin ve toplam deneme sayısı gösterilir.

Öne Çıkan Özellikler:
✔ Random sayı üretme
✔ Kullanıcı giriş kontrolü
✔ Sınırsız tahmin hakkı
✔ Deneme sayısı takibi
✔ Kullanıcı dostu uyarılar

Örnek Çalışma Mantığı:
Kullanıcı tahmin girer
Eğer küçük → “Sayı küçük”
Eğer büyük → “Sayı büyük”
Eğer doğru → “Tebrikler! Deneme sayısı: X”

Geliştirilebilir Özellikler:
Tahmin sayısı limiti eklenebilir
Aralık kullanıcıdan alınabilir
Arayüze butonlu tasarım eklenebilir
Puan sistemi eklenebilir
