# JavaScript Dosyaları

Bu klasörde web sayfasına etkileşim ve dinamik özellikler ekleyen JavaScript dosyaları bulunmaktadır.  

## Açıklama
- `script.js` ana JavaScript dosyasıdır.
- Buton tıklamaları, animasyonlar, veri kontrolü gibi işlemler burada yapılır.
- HTML dosyasında `<script src="js/script.js"></script>` ile bağlanır.

## Kullanım
1. HTML dosyaları ile birlikte çalıştırıldığında sayfada etkileşimler aktif olur.
2. Tarayıcı konsolunda hata kontrolü yapılabilir.
